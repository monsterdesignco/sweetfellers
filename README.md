# Sweet Fellers Tree Care — website

Static site (plain HTML/CSS + assets). No build step.

- Entry: index.html
- Pages: gallery.html, blog.html, shop.html
- assets/: images, SVGs, pages.css

## Deploy (Netlify)
Connect this repo; publish directory = repo root (see netlify.toml). No build command.

## Domain
sweetfellers domain is registered at Wix — point DNS at Netlify after first deploy.
